# Práctica primeiro repo en github

## Comandos empregados:

```bash
git init
```
> Initcialización do repositorio local.

```
git config --global user.name "Xian Durán"
Este comando sirve para asignar o nome  git

git config --global user.email "duran.taboada.xian.ald@gmail.com"
Poñemos un correo electronico o git

git config --list
Mostra as propiedades de git

git status 
mostra o estado do directorio da area do traballo


git init
Iniciamos o repositorio local nunha carpeta

git add . 
Poñemos en seguimento os arquivos da carpeta

git commit -m "Primer commit"


git diff
Da a ver as diferencias das orixes de datos de git 

git clone 
Sirve para clonar un repositorio remoto a local


git reset --hard
Sirve para desfacer cambios locais no estado de un repositorio 

git checkout --archivoejemplo.txt
Desfai os cambios realizados no ficheiro que selecionamos para volver a súa versión anterior.

git clean -f
Desfai os cambios ralizados para volver unha version do repositorio


git reset --soft HEAD~1
Desgacemos o último commit e mantemos cambios anterioes no directorio de traballo ea zona de intercambio temporal



git reset --hard HEAD~1
Desfai o último commit e os cambios anteriores do directorio de traballo volvendo a version anterior do repositorio


```
